# Script de Configuração do Ambiente de Desenvolvimento

Este é um script de configuração para criar meu ambiente de desenvolvimento nas distros baseadas em Ubuntu.

## Etapas

```
1. Atualização do Sistema e Instalação de Pacotes Básicos:

    Esta etapa atualiza o sistema e instala algumas ferramentas básicas como Git, Discord, Steam e Visual Studio Code.

2. Instalação do Node.js:

    Esta etapa instala o Node Version Manager (NVM), que permite a instalação do Node.js. Em seguida, o Node.js é instalado, e o pacote Vue CLI é globalmente instalado.

3. Instalação do Ruby:

    Esta etapa instala o Ruby junto com Rails.

4. Instalação do Rust:

    O script instala o Rust via Rustup, que é o gerenciador de versões do Rust.

5. Instalação do Golang:

    Instala o Go (Golang) no sistema.

6. Instalação do MySQL e PostgreSQL:

    Esta etapa instala os servidores de banco de dados MySQL e PostgreSQL, juntamente com as extensões contrib do PostgreSQL.

7. Instalação do DBeaver:

    Instala o DBeaver, uma ferramenta de gerenciamento de banco de dados, adicionando seu repositório e chave de assinatura.

8. Instalação do Insomnia:

    Instala o Insomnia, um cliente para testar APIs REST, adicionando seu repositório e chave de assinatura.

9. Instalação do Docker:

    Esta etapa instala o Docker, um sistema de containerização, adicionando o repositório e chave de assinatura do Docker.

10. Remoção do LibreOffice:

    Remove todos os pacotes relacionados ao LibreOffice.
```
